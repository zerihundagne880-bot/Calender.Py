### 2. Quadratic Equation Solver (`quadratic.py`)
This script solves quadratic equations in the form of $ax^2 + bx + c = 0$. 
* It calculates the **discriminant** ($D = b^2 - 4ac$).
* It identifies if the roots are **real**, **single**, or **complex**.
* It provides accurate results for math enthusiasts and students.
